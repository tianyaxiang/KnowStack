### 关于这个网站

这是一个基于 **Next.js + Contentlayer + GitHub API** 构建的现代化博客系统。

### 技术栈

- Next.js - React 框架
- Contentlayer - 内容管理
- Tailwind CSS - 样式
- Better Auth - 身份验证
- GitHub API - 内容存储
